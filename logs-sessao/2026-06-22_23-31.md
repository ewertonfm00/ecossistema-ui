## Decisões técnicas

- **Pure Date (sem libs):** Todos os helpers de calendário usam `Date` nativo + `Intl.DateTimeFormat` — nenhuma dependência adicionada
- **`new Date('YYYY-MM-DD')` é UTC:** Descoberto que string ISO sem hora é parseada como UTC pelo Node.js/V8. Em fusos negativos (UTC-3 = Brasil), `new Date('2026-06-01').getMonth()` retorna 4 (maio). Solução: sempre usar `new Date(year, month-1, day)` nos testes
- **React 18 event batching:** Dois `fireEvent.click` consecutivos podem bater em versão antiga do handler se não houver `rerender` explícito entre eles. Usar `rerender` com o state atualizado para garantir que o segundo clique veja o estado do primeiro
- **findInMonthCell helper:** Filtrar células por `!c.className.includes('text-neutral-300')` em vez de aria-label de locale — mais robusto para ambientes sem ICU completo
