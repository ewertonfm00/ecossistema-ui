## Decisões técnicas relevantes

| Decisão | Motivo |
|---------|--------|
| ReactDOM.createPortal sem mock | testing-library já encontra elementos no document.body |
| `Toast as ToastItem` no barrel | evita colisão de nome com Toast component de molecules/Toast |
| `SelectCustomOption` (não SelectOption) | colisão com SelectOption exportado pelo átomo Select nativo |
| Tooltip trigger="hover" usa CSS puro | backward compat com testes existentes que checam invisible/group-hover:visible |
| Ícone Bell no DatePicker | CalendarDays não estava registrado no iconMap do Icon atom |
| Math.random() como ID fallback | jsdom não garante crypto.randomUUID; geração determinística melhor para testes |
